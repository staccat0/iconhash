## 表格内图片显示调整

python的openpyxl库无法将图片嵌入到单元格内，所以需要手动调整格式将图片嵌入到表格中。WPS存在功能可以将图片嵌入到表格当中。

调整过程如下：

查找-->选择窗口-->选择所有对象后右键-->切换为嵌入单元格图片
